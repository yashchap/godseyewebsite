Overview of God's Eye

God’s Eye is a Mixed Reality based tool for vulnerability assessment for a person. It is an application that is deployed on Microsoft Hololens that allows a user to recognize a person’s face and overlays data mined from various public resources such as Facebook, LinkedIn etc. It gives the user an insight of their potential threats in the near future that could arise from camera equipped MR devices connected to AI powered cloud computing.

Our Goals

Ideas and concepts

Leverage all the exposed personal data on public websites of a person in a single platform.

Detect threats

Demonstrate an application that is futuristic which holds information of every recognized person and deliver a vulnerability score for every person by performing analytics on the information extracted from websites.

Future of product

Deploy the application on a mixed reality platform - Microsoft Hololens which can become a common device in every household.

Vision

Give an insight of how dystopian an application like this can be developed and misused in our lives.

Our features

Face Detection and Recognition

Identification of the target from the crowd, using Microsoft Azure Face API.

Vulnerability score analysis

Predicting vulnerability score of the target based on features like, annual income, number of dependents, recent check in etc.

Mining personal information

Mined personal imformation of the target from social media platform like, Facebook.

Mining interests information

Fetched what all possible interest of the target from their public profile available on the internet.

Mining financial information

Scrapped financial information like, work information, salary range, recent venmo transactions through public API for Linkedin, Glassdoor and Venmo.

Mining username information

Fetched all possible username of the target from various social platfrom like, Facebook, Instagram, Twitter, LinkedIn.

Mining connection information

Displays closest connection of the target on social media account like Facebook, Instagram.
